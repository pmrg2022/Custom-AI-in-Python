# Custom-AI-in-Python
Small, basic AI I made in Python, used to be a Swift Playground AI, it was not as good, so I moved it to Python, and it still has some Swift playground ideas in its training data, but that will be easy to fix.

---
* For running, just run generate.py

* For training, run tokenizer.py, then train.py
---

`Training.txt` is the old training data that was used in swift playground, but now it uses a `jsonl` file
